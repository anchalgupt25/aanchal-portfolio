# Aanchal Gupta — Product Portfolio

A standalone, responsive portfolio website for Aanchal Gupta.

## Site structure

- `index.html` — complete portfolio website
- `netlify.toml` — Netlify publishing configuration
- `.gitignore` — ignores local operating-system files

## Deploy with Netlify

1. Push this repository to GitHub.
2. In Netlify, choose **Add new project → Import an existing project**.
3. Select GitHub and choose this repository.
4. Leave the build command empty.
5. Set the publish directory to `.` if Netlify does not read it automatically from `netlify.toml`.
6. Deploy.

Any future commit pushed to the connected GitHub repository will trigger a new Netlify deployment.

## Local preview

Open `index.html` in a browser. No build tools or dependencies are required.
